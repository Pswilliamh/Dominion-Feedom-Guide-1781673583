# Dominion Freedom Pad

## Vision
A landscape tablet application (16:9) for critical operations and emergency response. Designed for authoritative clarity with large, touch-optimized interface elements. Users navigate between master modules (Daily Life, Transport, Financial, Security) while maintaining constant access to emergency functions.

## Design
**Color System:**
- `--navigation: 220 100% 20%` (navy) - Left dock, primary structure
- `--transport: 38 92% 50%` (amber) - Transport category
- `--financial: 142 76% 36%` (green) - Financial category
- `--security: 0 84% 40%` (crimson) - Security category
- `--emergency: 0 100% 50%` (bright red) - Emergency button
- `--background: 0 0% 100%` (white) - Clean base
- `--foreground: 220 25% 15%` (dark slate) - Text
- `--card-glow: 220 70% 50%` (soft blue glow) - Card borders
- `--muted: 220 15% 95%` (light grey) - Subtle backgrounds
- `--accent: 220 90% 56%` (bright blue) - Interactive highlights

**Typography:**
- Headings: Rajdhani (700, 600, 500) - geometric, authoritative
- Body: IBM Plex Sans (400, 600) - clean, readable
- Data/Labels: IBM Plex Mono (400, 500) - technical precision

**Style Direction:** Industrial control panel aesthetic - bold, clear, crisis-ready. Large touch targets, spacious layout, color-coded categories for instant recognition.

## Features
- **Navigation Dock:** Persistent left vertical bar with four master modules + emergency button
- **Command Matrix:** Central 3x2 grid of large interactive cards for category content
- **Communication Canvas:** Right panel with live message preview and receiver feedback windows
- **Emergency System:** Always-accessible RED button with high-visibility flashing animation
